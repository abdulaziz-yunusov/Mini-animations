# Mini Animations Library

This repository contains a collection of small, reusable animations that can be used in various projects.
In this project folder you can find some mini animations. Use any of them freely.

## Purpose

The goal of this repository is to provide simple, lightweight animations that can be easily integrated into web, mobile, or desktop applications. These animations are designed to save development time and serve as ready-to-use building blocks.

## Contents

* UI animations (buttons, loaders, transitions)
* Micro-interactions
* Decorative effects
* Experimental animation snippets

## Usage

You are free to:

* Copy and use any animation in your own projects
* Modify and adapt them to fit your needs
* Combine multiple animations

No attribution is required, but it is appreciated.

## Structure

Each animation is organized in its own folder and typically includes:

* Source files
* Minimal implementation example
* Any required assets

## Contribution

Contributions are welcome. If you have useful or creative animations:

1. Fork the repository
2. Add your animation in a separate folder
3. Submit a pull request

## Notes

* Animations are kept minimal and focused for easy integration
* No unnecessary dependencies unless explicitly stated
